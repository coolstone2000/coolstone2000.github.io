위는 그냥 처음 써두는 문법

# 1. 헤더

"#"은 소제목 - "#"개수가 늘어날 수록 점점 크기가 작아짐 (6개 까지)

# 2. 블록

">" 쓰면 아래로 블럭 하나하나 써짐

# 3. 목록

"1., 2., 3." 순서대로
"*" 는 동그라미, "-" 는 빈 동그라미, "+"는 채운 네모

# 4. 코드

"<pre>
<code>
{code}
</code>
</pre>"

를 사용하거나 

"```여기에 어떤 언어인지 쓰고
{code}
```"

# 5. 수평선

"---" 하면 된다

# 6. 링크

"사용문법: [Title](link)
적용예: [Google](https://google.com, "google link")"
를 쓰거나
"<>"를 쓰면 된다

# 7. 강조

"*single asterisks*
_single underscores_
**double asterisks**
__double underscores__
~~cancelline~~"
